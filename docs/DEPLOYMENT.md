# Deployment Guide for SAQR ERP

This document provides steps for deploying the SAQR ERP application on Railway and Vercel.

## Railway Deployment Steps
1. **Sign Up / Log In** to [Railway](https://railway.app/).
2. Click on `New Project`.
3. Select `Deploy from GitHub` and authorize Railway to access your GitHub repositories.
4. Choose the repository `aaa2333p/saqr-erp`.
5. Configure your environment variables as needed in the Railway dashboard.
6. Click on `Deploy`. Your application should start building and will be live shortly.

## Vercel Deployment Steps
1. **Sign Up / Log In** to [Vercel](https://vercel.com/).
2. Click on `New Project`.
3. Import the `aaa2333p/saqr-erp` repository from GitHub.
4. Configure the settings, including framework presets as per your application's needs.
5. Set up any necessary environment variables.
6. Click `Deploy` and wait for the deployment to complete. Your application will be live on Vercel.

---

> **Note:** Make sure all necessary configurations and environment variables are set for a successful deployment.
